# Model Performance Results

The models were evaluated based on classification accuracy.

| Model | Accuracy |
|------|---------|
| MobileNetV2 | ~77.23% |
| ResNet50 | ~68.99% |

Among the tested models, InceptionV3 achieved the highest test accuracy.

The system successfully classified animal breeds from images and provided additional breed information to the user.
