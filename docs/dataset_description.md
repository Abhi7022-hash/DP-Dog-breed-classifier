# Dataset Description

The dataset used in this project contains images of different animal breeds including dogs and cattle.

Total Classes: 111

The dataset is organized in folder structure format where each folder represents one breed class.

Each folder contains multiple images representing that breed.

Dataset Structure:

dataset1/
 ├── train/
 │    ├── breed1/
 │    ├── breed2/
 │    ├── breed3/
 │    └── ...
