# Animal Breed Classification System

This project is a deep learning based web application that classifies animal breeds from images.

The system allows users to upload an image or capture a photo using a camera. The image is processed using deep learning models such as InceptionV3, ResNet50, and MobileNetV2.

The application predicts the breed and displays additional information including breed features, habitat, and nature.

The system is designed to help farmers, veterinarians, and pet owners identify animal breeds easily.
